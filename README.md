# z3

**Experimental WIP** Crystal bindings for [Z3 Theorem Prover](https://github.com/Z3Prover/z3)

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     z3:
       github: bcardiff/crystal-z3
   ```

2. Run `shards install`
